Project Overview (개요): 
이 프로젝트는 내가 신입 개발자로써 포트폴리오를 만들기 위한 프로젝트야.
타겟 사용자는 모든 사람을 대상으로 하되 주 사용자는 채용 담당자겠지?
Tech Stack (기술 스택):  React.js , TypeScript, Tailwind, 상태 관리는 zustand, 라우팅은 react-router-dom(새로 업데이트된 버전으로 사용), 
Key Features (주요 기능): 
메인페이지(랜딩페이지), 프로젝트 리스트 페이지, 자기소개 페이지, 방명록 페이지
UI/UX Design Concept: 디자인은 모던, 미니멀리즘 , Flat하게 구현 폰트도 이에 맞는 톤앤매너로 구현.
Architecture & Folder Structure: Atomic Design, Feature-based
Data Flow & API State: 데이터 패칭은 axios, 데이터 관리는 tanstack query를 사용할꺼야. 디비는 supabase를 사용할꺼야.
Milestones (마일스톤): 개발 단계별 계획.
